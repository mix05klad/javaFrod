# Εβδομάδα 1: Καλώς ήρθατε στον κόσμο της Java!

Αυτή είναι η αφετηρία μας. Πριν αρχίσουμε να λύνουμε πολύπλοκα προβλήματα και να χτίζουμε δικά μας προγράμματα, πρέπει πρώτα να γνωριστούμε με τη γλώσσα και τα εργαλεία της.

## Λίγα λόγια για τη Java
Η Java δεν είναι απλά άλλη μια γλώσσα προγραμματισμού, είναι ένα ολόκληρο οικοσύστημα. Δύο είναι τα πιο σημαντικά πράγματα που κρατάμε από αυτή την πρώτη μας επαφή:

1. **"Write Once, Run Anywhere":** Αυτό είναι το μεγάλο της ατού (και το αρχικό της motto). Γράφουμε τον κώδικα μια φορά και, χάρη στη "Μηχανή" της Java (το περιβόητο **JVM** - Java Virtual Machine), το πρόγραμμά μας μπορεί να τρέξει σε Windows, Mac, Linux, χωρίς να αλλάξουμε ούτε κόμμα!
2. **Αντικειμενοστρέφεια:** Αν και στο πρώτο μάθημα αρκεστήκαμε στο να τυπώσουμε ένα απλό μήνυμα στην οθόνη, η Java είναι σχεδιασμένη γύρω από την έννοια των "Αντικειμένων". Όλα οργανώνονται σε Κλάσεις.

**Τι καλύψαμε στο πρώτο μας εργαστήριο:**
* Στήσαμε το περιβάλλον εργασίας μας (κατεβάσαμε το JDK και ρυθμίσαμε το IDE μας).
* Είδαμε τη διαδρομή που κάνει ο κώδικάς μας: από αρχείο κειμένου (`.java`) μεταφράζεται από τον Compiler στα ακαταλαβίστικα για εμάς, αλλά κατανοητά για το JVM, bytecodes (`.class`).
* Γράψαμε την κλασική πρώτη εντολή κάθε προγραμματιστή: `System.out.println("Hello World!");` και είδαμε πώς δουλεύουν οι βασικές μεταβλητές.
