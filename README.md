#  Αντικειμενοστρεφής Προγραμματισμός I (Java) - 2026

Καλώς ήρθατε στο επίσημο αποθετήριο (repository) του μαθήματος!

Εδώ θα βρίσκετε συγκεντρωμένο όλο το απαραίτητο υλικό για την προσωπική σας μελέτη και την προετοιμασία σας για τα φορντιστήρια. Το υλικό θα ανανεώνεται **κάθε εβδομάδα** μετά την ολοκλήρωση των φροντιστηρίων.

##  Τι περιλαμβάνει αυτό το Αποθετήριο;

Κάθε εβδομάδα θα προστίθεται ένας νέος φάκελος (π.χ. `frod-00`, `frod-01`) ο οποίος θα περιέχει:
*  **Διαφάνειες:** Το θεωρητικό υλικό του μαθήματος.
*  **Κώδικας (Project_Starter & Project_Solution):** Τον αρχικό σκελετό των ασκήσεων καθώς και τις ολοκληρωμένες λύσεις που γράψαμε στην τάξη.
*  **Quizzes:** Μέσα στο αρχείο `README.md` της κάθε εβδομάδας, θα βρίσκετε το link για το αντίστοιχο εβδομαδιαίο Quiz στο Google Forms.

##  Ο Απόλυτος Οδηγός: Πώς να κατεβάσετε το υλικό

Επειδή ο κώδικας και το υλικό θα αυξάνονται κάθε εβδομάδα, **δεν χρειάζεται να κατεβάζετε τα πάντα από την αρχή κάθε φορά**. 

Υπάρχουν 2 τρόποι για να συγχρονίζετε τα αρχεία στον υπολογιστή σας. Επιλέξτε αυτόν που σας βολεύει καλύτερα:

### Επιλογή 1: Με τη χρήση του terminal (Η σωστή λύση)
Αυτός είναι ο τρόπος που δουλεύουν οι επαγγελματίες προγραμματιστές! Αν θέλετε να μάθετε το Git σωστά από την αρχή:

**Την 1η φορά (Λήψη όλου του υλικού - Clone):**
1. Ανοίξτε το τερματικό σας (Command Prompt ή PowerShell στα Windows, Terminal σε macOS/Linux).
2. Πλοηγηθείτε στον φάκελο που θέλετε να αποθηκευτεί το μάθημα χρησιμοποιώντας την εντολή `cd` (π.χ., `cd Documents`).
3. Τρέξτε την παρακάτω εντολή:
   ```bash
   git clone https://github.com/mix05klad/javaFrod.git
4. Ένας νέος φάκελος javaFrod θα δημιουργηθεί αυτόματα με όλο το υλικό.

**Τις επόμενες εβδομάδες (Λήψη μόνο των νέων αρχείων - Pull):**

**Προσοχή:** ΜΗΝ τρέξετε ξανά την εντολή clone!

1. Ανοίξτε ξανά το τερματικό σας.
2. Μπείτε μέσα στον φάκελο του μαθήματος που είχατε κατεβάσει την πρώτη φορά:
```bash
cd javaFrod
```
3. Κατεβάστε τις νέες αλλαγές της εβδομάδας τρέχοντας απλά:
```bash
git pull
```
4. Έτοιμοι! Τα νέα αρχεία κατέβηκαν αστραπιαία στον υπολογιστή σας.

### Επιλογή 2: Download ZIP
Αν κάτι πάει στραβά με τις παραπάνω επιλογές, αν βιάζεστε ή αν βρίσκεστε σε υπολογιστή της σχολής και θέλετε απλώς τα αρχεία:

1. Πηγαίνετε στην κορυφή αυτής της σελίδας στο GitHub.
2. Πατήστε το πράσινο κουμπί που γράφει **"<> Code"**.
3. Επιλέξτε **"Download ZIP"**.
4. **Σημείωση:** Με αυτή τη μέθοδο, κάθε εβδομάδα θα κατεβάζετε *ξανά* ολόκληρο το υλικό του εξαμήνου από την αρχή σε μορφή συμπιεσμένου φακέλου (.zip). Θα πρέπει να τον κάνετε αποσυμπίεση (extract) και να οργανώνετε τους φακέλους σας χειροκίνητα.
